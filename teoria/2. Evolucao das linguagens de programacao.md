## Século XIX
- 1801 - Tear de Jacquard
	- Criou uma máquina de tear controlada por cartões perfurados, cada furo representava uma instrução
- 1837 - 1843 - Máquina analítica + Ada Lovelace
	- Charles Babbage projetou o primeiro computador conceitual (máquina analítica). Já a Ada Lovelace criou o primeiro algoritmo da história e percebeu que máquinas podem manipular qualquer símbolo não só números
## Século XX
- Década de 1940 - nascimento dos algoritmos e a teoria
	- Plankalkül (nome da linguagem) (1942-1945): 
		- Konrad Zuse (Alemanha) criou a primeira linguagem e programação formal porém ela só foi reconhecida em 1972
	- Arquitetura de von Neumann (1945):
		- é o modelo fundamental de computadores modernos, caracterizada por armazenar dados e instruções no mesmo espaço de memória
	- Alan Turing: 
		- Pai da ciência da computação
		- Máquina de Turing: Modelo teórico provando que uma máquina poderia simular qualquer lógica algorítmica 
	- Os cientistas estavam inventando o conceito de algoritmos e como uma máquina pode "entender" instruções
- Década de 1950 - Inicio das linguagens de alto nível
	- Linguagem de máquina: 0s e 1s
	- Em 1951 surgiu o Assembly (ainda de baixo nível porém mais legível que a linguagem das máquinas)
	- Primeiras linguagens de alto nível (mais próximas da linguagem humana):
		- Fortran(1954): Usada para cálculos científicos e engenharia
		- LISP: Inteligência artificial (listas e recursão)
		- Cobol: Sistemas bancários e de negócios
- Décadas de 1960-1970 - A era de ouro
	- Algol 60 (1960): Estrutura moderna, blocos de código, escopo de variáveis. Influenciou tudo que veio depois
	- BASIC (1964): Criado para "qualquer pessoa programar", permitiu que pessoas comuns (não apenas cientistas) começassem a programar. Bill Gates e Paul Allen fizeram o primeiro produto da Microsoft usando essa linguagem
	- Pascal(1970): Sintaxe clara (o código deve ter começo, meio e fim claros usando procedimentos e funções para evitar repetições). Foco em estrutura
	- C(1972): Eficiência de baixo nível com legibilidade, desenvolvida para a programação do sistema Unix. É a base de tudo que usamos hoje (C++, Java, Python)
	- Prolog: Também é dessa época 
- Década de 1980 - Objetos e comercialização
	- C++(1983): C com classes, adicionou orientação a objetos em C (herança, polimorfismo, classes, encapsulamento)
	- Objective-C(1983): Ampliação do C, é utilizada para desenvolvimento de software para sistemas Apple macOS e IPhone/IPad
- Década de 90 
	- Python(1989): Foco em simplicidade e legibilidade, sintaxe limpa. Data Science, IA, automação, web
	- Lua(1993): Leve, embarcável e versátil, Lua é amplamente utilizada como linguagem de script em jogos, sistemas embarcados e aplicações que necessitam de extensibilidade
	- Java(1995): A ideia central era produzir uma ferramenta capaz de ser utilizada independentemente de um dispositivo eletrônico específico o famoso princípio "write once, run anywhere" (escreva uma vez, execute em qualquer lugar)(executar em qualquer ambiente por meio do Java Virtual Machine (JVM)). Principal uso: aplicativos para Android
	- Ruby, JavaScript, PHP, C#
## Resumo
- 1801 – Cartões perfurados (Jacquard)
- 1843 – Primeiro algoritmo (Ada Lovelace)
- 1945 – Arquitetura von Neumann
- 1954 – FORTRAN (primeira alto nível comercial)
- 1958/59 – LISP e COBOL (IA e negócios)
- 1960 – ALGOL (estrutura moderna)
- 1964 – BASIC (programação para todos)
- 1972 – C (a “mãe” das linguagens atuais)
- 1983 – C++ (orientação a objetos)
- 1989–1995 – Python, Java, JavaScript (era da web e da produtividade)
