# Paradigmas de Programação (S01) 🧠

Repositório de estudos e práticas da disciplina S01 do **Inatel**.
## 📂 Organização
- **Teoria:** Anotações e conceitos das aulas teóricas
- **Prática:** Aulas do laboratório
	    - **Relatórios:** Relatórios e exercícios das aulas em laboratório
		- **Sintaxe:** Sintaxe básica das linguagens usadas nos relatórios
## 🚀 Linguagens Utilizadas
Até o momento, o repositório tem as seguintes linguagens:
1. **Basic**
2. **Lua**
3. **Rust**
4. **Go**
